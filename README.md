# InAir

InAir is a mobile app that allows you to consult the air quality based on the current geographic location of your phone, in a simple, practical, and modern way. The unique thing about InAir is that it offers the user special recommendations: each user can create a personal profile, providing some basic and medical information in exchange for InAir's appropriate protective measures suggestions, especially for people with conditions like respiratory diseases, for example.

Developed in Flutter, InAir retrieves data from the OpenAQ public API, which provides extensive information about the Air Quality Index (AQI) worldwide.

Users are *not* required to create an account to use InAir, but it is certainly required if one wants to make the best use of it. The simplified version of the app offers the Air Quality Index (AQI) and a general recommendation. However, when creating an account and providing some medical information, such as diseases, allergies, and sensitivities, the app customizes its recommendations based on the user's profile. Additionally, InAir offers more features, such as the ability to search and save new locations.
